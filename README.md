# The Odin Tic Tac Toe

Aplicação de jogo da velha desenvolvido com HTML, CSS e JavaScript puro (vanilla JS).

## Descrição

Este projeto funciona como uma Single Page Application (SPA) simples jogar um jogo da velha (em inglês, tic tac toe).

O usuário pode:

- Jogar o jogo da velha contra outro usuário.

## Funcionalidades Implementadas

- Ainda serão desenvolvidas.

## Tecnologias e Conceitos Aplicados

- HTML5 semântico (`header`, `aside`, `main`, `footer`).
- CSS Grid e Flexbox para layout e alinhamento.
- Abordagem Mobile First com media query para telas maiores.
- Manipulação intensa do DOM com JavaScript vanilla.

## Estrutura do Projeto

- `index.html`: estrutura base da aplicação.
- `styles.css`: estilos, responsividade e visual dos componentes.
- `script.js`: lógica da aplicação.

## Como Executar

1. Clone este repositório.
2. Abra a pasta do projeto.
3. Execute o arquivo `index.html` no navegador.
